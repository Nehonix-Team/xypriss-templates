## Getting Started

### Prerequisites
- [XFPM](https://github.com/Nehonix-Team/xfpm) (XyPriss Package Manager)
- Node.js (Latest LTS recommended)

### Installation
```bash
xfpm install
```

### Development
Start the development server with live reloading:
```bash
xfpm run dev
```

### Production
Build and start the production instance:
```bash
xfpm run build
xfpm run start
```

## Project Structure
- `src/server.ts`: Application entry point.
- `src/configs/`: Configuration store and system variables.
- `src/router/`: Global traffic distribution logic.
- `src/routes/`: Service-specific business logic.
- `src/servers/`: Isolated server instance definitions.

## Multi-Server Orchestration (XMS)
This template is configured for **Multi-Server** mode, allowing multiple isolated nodes to run within a single process.
- **Auth Node**: Managed identity and authentication.
- **Main Node**: Primary API business logic.
